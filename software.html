Artifact Overview

This enhancement focused on improving the modular architecture and maintainability of the HELIOS server. I restructured the code into services and routers using Express.js, introduced logging middleware, and ensured separation of concerns across modules.

Project Background

HELIOS is my custom-built self-hosted media and request management server running on an UNRAID-powered NAS system. I implemented this project to automate request handling, logging, and media indexing using services like Plex, Sonarr, and a custom-built request interface.

Enhancement Highlights

  • Restructured HELIOS request logger using modular Express routes and middleware
  • Improved readability and maintainability by adopting single responsibility principles
  • Introduced centralized error handling and debug logging to aid future troubleshooting
  • Clean separation of frontend/backend behavior to isolate service responsibilities

Preview the Code

Below is a snippet from the refactored Express.js app used to register services and middleware: 

const express = require('express');
const requestLogger = require('./middleware/requestLogger');
const queueRouter = require('./routes/queue');

const app = express();
app.use(express.json());
app.use(requestLogger);
app.use('/queue', queueRouter);

module.exports = app;
