Portfolio Overview

This portfolio includes a selection of original and enhanced artifacts that demonstrate my competency across three critical categories:

  • Software Design & Engineering: The Software page highlights enhancements made to the backend routing logic of my HELIOS Express server, emphasizing modularity, scalability, and middleware integration.
  • Algorithms & Data Structures: The Algorithms page demonstrates an optimized media request queue using Python’s `heapq`, emphasizing duplicate handling and request prioritization.
  • Databases: The Databases page showcases a robust logging mechanism added to track user media requests persistently via structured logging and rotating logs.

Each artifact is paired with a reflection narrative explaining the enhancement process, challenges overcome, lessons learned, and alignment with program outcomes. These enhancements collectively fulfill all course outcomes, including:

  • Employing collaborative and contextual code review strategies
  • Communicating technical ideas clearly across written, oral, and visual forms
  • Designing solutions based on efficient data structures and trade-off analysis
  • Applying industry-standard practices and tools
  • Addressing security vulnerabilities and integrating safeguards into design
