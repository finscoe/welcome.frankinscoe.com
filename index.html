Welcome

Hello! I'm Frank Inscoe, an educator and lifelong learner pursuing my degree in Computer Science with a concentration in Data Analytics. This portfolio is the culmination of my capstone project for CS-499 at Southern New Hampshire University. It highlights my skills in backend engineering, database design, and algorithm optimization.

For my final project, I created HELIOS, a personal self-hosted media and request management server that integrates open-source services like Nextcloud, Plex, Jellyfin, and others using an UNRAID-powered NAS system. This system handles request logging, queuing, and media indexing, and has been engineered for modularity, maintainability, and future growth.

I invite you to explore the sections of this site to learn how I applied industry-standard practices across software design, data structures, and databases. You can also watch a narrated code review that walks through HELIOS and key implementation decisions.
