Welcome to My Capstone Portfolio

My name is Frank Inscoe, and after over a decade in higher education as a Spanish and English instructor, I began a new journey into the world of computer science. This capstone portfolio showcases my transition into technical roles focused on backend development, infrastructure engineering, and data-driven applications. It is the culmination of several years of study, self-teaching, and hands-on experimentation with systems like my self-hosted UNRAID-based server, HELIOS.

The HELIOS Project

HELIOS is my custom-built home server environment designed to securely host file sharing, media streaming, subscription tracking, and more — all powered by Docker and managed via UNRAID. It runs over 20 Docker containers and stores more than 276 TB of data across 27 drives. The landing page and account request form allow users to gain secure access to services like Nextcloud, Plex, Jellyfin, Overseerr, and Wallos.

Throughout the CS-499 course, I enhanced the HELIOS project in three ways:

  • Software Design & Engineering: Developed a modular Express.js backend and integrated service request form.
  • Algorithms & Data Structures: Created a priority queue in Python with SHA-256 deduplication for managing media requests.
  • Databases: Built an SQLite-backed logging system to persist and secure user-submitted requests.
