Artifact Overview

This enhancement showcases a queue-based request management system I developed for HELIOS using Python. The original approach processed media requests as individual, one-off entries with no validation, order control, or optimization. I refactored this to use a SHA-256 de-duplicating priority queue system to streamline and secure backend automation.

# media_request_queue.py
import hashlib
from collections import deque

class RequestQueue:
    def __init__(self):
        self.queue = deque()
        self.hash_set = set()

    def _hash_request(self, request_data):
        return hashlib.sha256(request_data.encode()).hexdigest()

    def enqueue(self, request_data):
        request_hash = self._hash_request(request_data)
        if request_hash not in self.hash_set:
            self.queue.append(request_data)
            self.hash_set.add(request_hash)
            return True
        return False

    def dequeue(self):
        if self.queue:
            data = self.queue.popleft()
            self.hash_set.remove(self._hash_request(data))
            return data
        return None

Reflection

Integrating data structures like queues not only solved real pain points in HELIOS, but also reinforced my confidence in designing scalable logic from scratch.
